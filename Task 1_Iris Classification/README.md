Task 1: Iris Flower Species Classification

Directory: Task1_Iris_Classification

🎯 Objective

Develop an end-to-end supervised machine learning pipeline to classify Iris flower species based on four morphological measurements.

📊 Dataset Features
Sepal Length
Sepal Width
Petal Length
Petal Width
🛠 Pipeline
Data preprocessing
Feature normalization using StandardScaler
Logistic Regression classifier
Model evaluation using:
Accuracy
Precision
Recall
Confusion Matrix
✅ Results
Classification Accuracy: 96.67%
Model serialized using Joblib
Deployment-ready model:
iris_logistic_regression_model.pkl
